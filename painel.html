Configuração Inicial

Conecte seu Google Apps Script para começar

1

Abra o Google Apps Script

Clique no botão abaixo. Vai abrir uma nova aba no seu navegador.

Abrir Google Apps Script

O que fazer lá:

👉 Na tela que abrir, clique em "Novo projeto" (botão azul no canto superior esquerdo)

👉 Vai aparecer um editor com um código. Selecione tudo (Ctrl+A) e apague

2

Substitua o código que aparece no editor

Você vai ver isso no Apps Script:
function myFunction() {
  // ← esse código aqui
}

O que fazer:

1️⃣

Clique em qualquer lugar dentro da área de código (na área escura com o texto)

2️⃣

Pressione Ctrl + A para selecionar tudo

3️⃣

Pressione Delete (ou Backspace) para apagar

4️⃣

Clique no botão abaixo para copiar o código do EVENTA, depois volte no Apps Script e pressione Ctrl + V para colar

5️⃣

Pressione Ctrl + S para salvar — se pedir nome do projeto, escreva EVENTA e clique OK

3

Publique o código (Deploy)

Siga exatamente estes passos:

1️⃣

No Apps Script, clique no botão azul "Implantar" (no canto superior direito da tela)

2️⃣

Clique em "Nova implantação" no menu que aparecer

3️⃣

Ao lado de "Tipo", clique na engrenagem ⚙️ e selecione "App da Web"

4️⃣

Em "Executar como" deixe: Eu mesmo

5️⃣

Em "Quem tem acesso" escolha: Qualquer pessoa

6️⃣

Clique em "Implantar" (botão azul)

7️⃣

Se aparecer uma tela pedindo autorização, clique em "Autorizar acesso", escolha sua conta Google e clique em "Permitir"

4

Copie e cole o link gerado aqui

Onde achar o link:

👉 Após o passo anterior, vai aparecer uma caixa com "URL do app da Web"

👉 Clique no botão "Copiar" ao lado dessa URL

👉 Cole no campo abaixo 👇

⚠️ URL inválida. Verifique se copiou o link completo que termina com /exec
🔒 Seus dados ficam 100% no seu Google Drive.
Nenhuma informação é enviada para outros servidores.
